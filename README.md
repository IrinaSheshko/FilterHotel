# NYC Hotels React Project

Простой интерактивный список лучших отелей Нью-Йорка, сделанный на **React**. Проект демонстрирует работу с компонентами, хуками и современным UI.

## 🔹 Функционал

- Показ списка отелей с изображениями и описанием  
- **Show more / Show less** для длинных описаний  
- Удаление отдельной карточки (`Remove`)  
- Очистка всего списка (`Clear All`)  
- **Поиск по названию отеля** (динамический фильтр)  
- Анимация появления карточек и hover-эффекты  

## 🔹 Технологии

- React + Hooks (`useState`)  
- CSS для стилизации и анимации  
- Компонентная структура (`App`, `HotelList`, `HotelCard`)
