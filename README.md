# 🏨 NYC Top Hotels List

Интерактивное приложение для поиска и управления списком лучших отелей Нью-Йорка. Проект создан на **React** и демонстрирует работу с динамической фильтрацией данных.

## 🚀 [Live Demo](https://IrinaSheshko.github.io/nyc-hotels-list)

---

### 🌟 Основные функции:
* **Динамический поиск:** Мгновенная фильтрация отелей по названию по мере ввода текста.
* **Управление списком:** Возможность удалять отдельные карточки отелей или очистить весь список одной кнопкой.
* **Счетчик отелей:** Заголовок автоматически обновляется, показывая актуальное количество отелей в списке.
* **Адаптивный дизайн:** Интерфейс корректно отображается на мобильных устройствах, планшетах и десктопах.

---

### 🛠 Стек технологий:
* **React** (Hooks: `useState`)
* **JavaScript** (ES6+)
* **CSS3** (Flexbox, Media Queries)
* **GitHub Pages** (для деплоя)
